버전 1.1 [최신버전] 다운로드 페이지로 이동

- 보안을 위해 비밀번호가 추가되었습니다.

- 텍스트 복사, 학생 정보 수정, 학생 삭제 등의 각종 편의성 기능이 개선되었습니다.

- 잘못 정렬 순서가 반전되었습니다.

- 잘못이 많아져도 아래로 스크롤이 되지 않던 오류가 수정되었습니다.

- 장난이 추가되었습니다.
